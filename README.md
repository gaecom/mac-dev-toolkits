mac-dev-toolkits
================
### Jumpcut

Jumpcut 是系统剪贴板的扩展，无主界面。它能且只能记住你放入剪贴板的文本，可化身剪刀图标常驻 Mac 系统菜单，单击激活（上图）；或快捷键调用弹出界面 (bezel)，默认快捷键为 control+option+V

下载地址:http://jumpcut.sourceforge.net/
### iterm2

ierm2是一个更好替代Mac自带终端与iterm的软件

- 屏幕分割
- 选中即复制
- 系统热键
- 全屏
- 保存当前快照

下载地址:http://www.iterm2.com/
